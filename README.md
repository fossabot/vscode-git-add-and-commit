## Usage

### Add all edited files to Git and commit them

1. Hit **Ctrl + Shift + A** (PC) / **Cmd + Shift + A** (Mac)
2. Enter the commit message.
3. Press **ENTER**.

### Add ONLY the current file to Git and commit it

1. Hit **Ctrl + Shift + Z** (PC) / **Cmd + Shift + Z** (Mac)
2. Enter the commit message.
3. Press **ENTER**.
